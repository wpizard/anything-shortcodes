# Changelog

All notable changes to this project will be documented in this file.

## [1.4.0] - 2025-11-04
- Added support for pipe-separated dynamic function arguments in the `[anys]` shortcode.
- Added shortcode support in navigation menu **Label** and **Link** fields.
- Added the `anys_date_i18n_jalali` function to support Jalali date formatting.

## [1.3.0] - 2025-10-07
- Added Link type.

## [1.2.0] - 2025-09-29
- Added Term Field type.

## [1.1.0] - 2025-08-15
- Added Function type.
- Added Settings page.
- Added Whitelisted Functions setting for better security control.
- Added more formats (json, serialize, unserialize, print_r, var_export, implode, keys, capitalize, uppercase, lowercase, strip_tags, values, keys_values).
- Improved shortcode registration for future shortcodes.
- Improved hooks naming conventions.
- Improved docs.

## [1.0.0] - 2025-08-04
- Initial release.
